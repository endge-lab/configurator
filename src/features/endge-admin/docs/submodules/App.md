# Endge.app

Модуль `Endge.app` хранит метаданные прикладного слоя: текущий проект, его идентификатор и связанный runtime-контекст.

Основная роль:
- единая точка доступа к текущему проекту;
- подготовка данных для UI/бизнес-модулей;
- использование в слоях orchestration без прямой привязки к деталям домена.
