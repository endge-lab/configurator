# DateTime

Компонент `<DateTime>` используется для отображения даты/времени с поддержкой таймзоны и собственного формата. Умеет автоматически интерполировать данные, полученные из контекста (например, через выражение `{{ ... }}`).

## Атрибуты

| Атрибут   | Тип     | Описание                                                        | Пример                      |
|-----------|---------|------------------------------------------------------------------|-----------------------------|
| `value`   | string  | Выражение или переменная, содержащая ISO-дату                   | `{{ user.createdAt }}`     |
| `timezone`| string  | Таймзона для отображения даты                                   | `"Europe/Moscow"`          |
| `format`  | string  | Строка формата отображения (см. [date-fns форматирование](https://date-fns.org/v2.30.0/docs/format)) | `"dd MMM yyyy, HH:mm"`     |

## Особенности

- Если дата указана некорректно, выводится `(invalid date)`.
- По умолчанию используется таймзона `UTC` и формат `yyyy-MM-dd HH:mm`.

## Пример использования

```jsx
<DateTime
  :value="order.createdAt"
  timezone="Europe/Moscow"
  format="dd MMM yyyy, HH:mm"
/>
