## Text

Компонент `<Text>` используется для стилизации и отображения текста в DSL-описаниях. Он поддерживает различные атрибуты для управления внешним видом текста: цвет, размер, фон, преобразования регистра, а также дополнительные CSS-классы.

## Поддерживаемые атрибуты

| Атрибут      | Описание                                                 | Пример значения       |
|--------------|----------------------------------------------------------|------------------------|
| **`bold`**      | Делает текст жирным                                      | `true` (без значения)  |
| **`italic`**    | Делает текст курсивом                                    | `true`                 |
| **`underline`** | Подчеркивание текста                                     | `true`                 |
| **`strike`**    | Зачёркивание текста                                      | `true`                 |
| **`color`**     | Цвет текста                                              | `#d32f2f`, `red`       |
| **`bg`**        | Цвет фона текста                                         | `#f0f0f0`              |
| **`size`**      | Размер шрифта                                            | `14`, `1rem`           |
| **`align`**     | Выравнивание текста                                      | `left`, `center`       |
| **`weight`**    | Толщина шрифта                                           | `400`, `bold`          |
| **`uppercase`** | Преобразует текст в ВЕРХНИЙ РЕГИСТР                      | `true`                 |
| **`lowercase`** | Преобразует текст в нижний регистр                       | `true`                 |
| **`capitalize`**| Первая буква -  заглавная                                 | `true`                 |
| **`class`**     | Дополнительные CSS-классы                                | `my-text-class`        |

Атрибуты можно свободно комбинировать для достижения желаемого эффекта.

---

## Пример использования

```jsx
<Text bold color="#d32f2f" size="16" underline>
  Привет, {{ user.name }}
</Text>_
