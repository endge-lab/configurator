# Общие сведения

При генерации HTML из JSX фреймворк автоматически добавляет к элементам DOM классы и атрибуты с префиксом `endge-*`. Ниже перечислены все классы и атрибуты, которые можно использовать в CSS для стилизации и точечного выбора элементов.

---

## Теги (элементы разметки)

К каждому тегу в JSX-дереве добавляется один класс и один атрибут. Используйте эти имена в селекторах.

| Элемент | Добавляемый класс | Добавляемый атрибут | Пример селектора в CSS |
|--------|--------------------|---------------------|-------------------------|
| Тег верхнего уровня (прямой потомок корня DSL) | `endge-tag-root` | `endge-tag-identity="<id>"` | `.endge-tag-root { ... }` или `[endge-tag-identity="..."] { ... }` |
| Все остальные теги (вложенные, в т.ч. внутри `v-if` / `v-else`) | `endge-tag` | `endge-tag-identity="<id>"` | `.endge-tag { ... }` или `[endge-tag-identity="..."] { ... }` |

**Итог по тегам:** у корневого тега всегда класс **`endge-tag-root`**, у всех остальных - **`endge-tag`**. У любого тега есть атрибут **`endge-tag-identity`** с уникальным значением в рамках дерева.

---

## Компоненты из домена

Обёртка компонента получает класс и два атрибута: для отображения (identity, может меняться) и стабильный id для селекторов стилей.

| Элемент | Добавляемый класс | Добавляемые атрибуты | Пример селектора в CSS |
|--------|--------------------|------------------------|-------------------------|
| Обёртка компонента из домена | `endge-component` | `endge-component-identity="<identity>"`, `data-endge-component-id="<id>"` | `.endge-component { ... }` или `[data-endge-component-id="abc123"] { ... }` |

**Итог по компонентам:** класс **`endge-component`**; **`endge-component-identity`** - человекопонятный identity (может меняться); **`data-endge-component-id`** - стабильный id сущности. В стилях и в JSON стилей используйте **id** и селектор **`[data-endge-component-id="..."]`**, чтобы привязка не ломалась при смене identity.

---

## Константы в коде

Имена вынесены в `@endge/core`:

- **EndgeJsxAttr** (`domain/types/jsx.types.ts`): `TagRoot`, `Tag`, `TagIdentity`, `ComponentIdentity`, `ComponentId` → `data-endge-component-id`.
- **Prefix.ComponentWrapper** (`domain/types/types.ts`): `endge-component`.

В стилях используйте классы и атрибуты из таблиц выше; для точечной привязки к компоненту - **`[data-endge-component-id="<id>"]`**.
