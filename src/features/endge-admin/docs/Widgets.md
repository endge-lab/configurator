# Виджеты

Краткое описание виджетов панели админки Endge.

| Виджет | Назначение |
|--------|------------|
| **Проект** | Дерево домена: компоненты, запросы, фильтры, виды, типы и др. Создание, перетаскивание, открытие документов. |
| **Инспектор** | Свойства и редактор текущего открытого документа (компонент, запрос, фильтр, вид и т.д.). |
| **Версии** | Список версий (снимков) домена, сохранение и восстановление. |
| **Настройки** | Редактирование глобальных настроек приложения. |
| **Ошибки** | Список ошибок компиляции и валидации домена. |
| **Локальное хранилище** | Просмотр и очистка данных в localStorage, связанных с Endge. |
| **Тестирование** | Режим и утилиты тестирования. |
| **События** | Лог событий шины Endge.events. |
| **Отладка** | Подключение вкладки к каналу отладки, выбор вкладки для отладки. |
| **Пульс** | Краткий sidebar-обзор runtime-сущностей. Детальная mock-картина открывается в singleton-вкладке «Пульс». |
| **Raph** | Инструменты по Raph (таблицы, запросы, сценарии на уровне рантайма). |
| **Документация** | Навигация по разделам документации и просмотр статей. В разделе «Roadmap» — планы по модулю диагностики, обработке ошибок, конфигурации и feature flags, RBAC и аудиту, уведомлениям, регистру модальных окон, хранению настроек по сессии, переопределению переменных через env, health checks, бэкапам и политикам восстановления, версионированию и обновлениям, виртуализации списков и таблиц, оптимизации бандла и изоляции tenant, доступности (a11y). |
| **Агент** | Панель агента (AI-помощник). |
| **Демонстрация** | Превью таблицы или данных из раздела «Помощь» инспектора. |

Виджеты можно перемещать в левую/правую/нижнюю панель или в плавающее окно (где поддерживается).
