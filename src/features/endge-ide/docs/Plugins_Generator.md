# Генератор

**Генератор** - плагин админки для быстрого создания набора сущностей домена: фильтра, запроса, компонента-таблицы и при необходимости вида.

## Назначение

По описанию полей (identity, отображаемое имя, пример значения, признак «в фильтр») одним нажатием создаёт в домене:

- **Фильтр** - с полями по отмеченным строкам (статический список, multiple).
- **Запрос** - REST с mock-данными по полям и привязкой к созданному фильтру.
- **Компонент-таблицу** - колонки по полям с компонентом отображения «text», привязка к запросу через переменную `items`.
- **Вид** (опционально) - связывает созданные компонент, фильтр и запрос; при включённом «Унаследовать сущности» сущности помечаются как унаследованные и в `meta.inheritedFrom` записывается ссылка на вид.

Идентификаторы сущностей имеют вид `gen-<view|component|query|filter>-<random>`.

## Как пользоваться

- Задать название и таблицу полей (при необходимости - «Тестовое заполнение»).
- Включить нужные галочки: генерация запроса, компонента, фильтра, вида, унаследование сущностей.
- Нажать «Сгенерировать». После генерации при создании вида открывается вкладка вида и закрывается вкладка генератора.

Для таблицы в домене должен быть компонент с identity `text`; иначе генерация компонента пропускается.
