# Policy (Политика)

Сущность домена, описывающая политику доступа или поведения. Коллекция Payload: `policies`.

**Поля:** `identity`, `displayName`, `description`, `folder`.

В виджете домена отображаются в секции «Политики» (root-policies). Используются для настройки правил и политик в приложении.
