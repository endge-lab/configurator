# Environment (Окружение)

Сущность домена, описывающая окружение выполнения (например `dev`, `prod`). Коллекция Payload: `environments`.

**Поля:** `identity`, `displayName`, `folder`, `isSystem`.

Системные окружения (`isSystem: true`) нельзя редактировать и удалять в редакторе. В виджете домена отображаются в секции «Окружения» (root-environments). Используются для переключения контекста (например текущее окружение в шапке приложения).
