# Component (legacy document)

Legacy-сущность хранит старую структуру Table/DSL-компонента: поля, колонки, bindings и JSX source. Она нужна как data-only документ для просмотра и редактирования исторической конфигурации.

Этот документ не компилируется, не создаёт runtime host и не участвует в UI rendering. Для исполняемого интерфейса используется `ComponentSFC`.
