# NOVA: обзор

`@endge/nova` - библиотека для Canvas-отрисовки и сценового управления.

На практике NOVA удобна, когда нужно:

- рисовать собственный UI на canvas;
- контролировать порядок фаз (`update -> matrix -> render -> flush`);
- работать со слоями (surface) и интерактивными узлами;
- держать стабильный FPS и управляемый lifecycle.

## Что внутри

- `NovaGraph` - фазовый граф обновлений;
- `NovaApp` - приложение/движок сцены;
- `NovaSurface` - слой отрисовки;
- `NovaNode` - узел сцены;
- `NovaRenderer2D`/`NovaRendererWebGL` - рендереры;
- `NovaCanvas`/`NovaGraphics` - канвас и графические утилиты;
- `Telemetry` - метрики графики и диагностика.

## Когда выбирать NOVA

- нужны кастомные визуализации, timeline, canvas-редакторы;
- важен контроль над рендер-пайплайном;
- хочется разделить данные (`@endge/raph`) и отрисовку (NOVA).

## Рекомендуемая связка

- данные и реактивность: `@endge/raph`;
- визуальный слой и интерактивность: `@endge/nova`.

Это даёт чистую архитектуру: изменения в данных создают `dirty`, а NOVA делает перерисовку в своих фазах.
