# AppCore

`AppCore` - прикладная точка запуска конфигуратора. Она не является федерацией и не хранит собственные модули.

В `AppCore`:
- создается `EndgeBootContext`;
- один раз подключается Vue-plugin через `Endge.use(EndgeVuePlugin)`;
- запускается ядро через `Endge.boot(ctx)`;
- хранится локальный флаг, чтобы не запускать конфигуратор повторно;
- `reset()` сбрасывает `Endge` и локальный флаг инициализации.
