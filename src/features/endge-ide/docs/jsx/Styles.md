# Общие атрибуты отступов (Margin / Padding)

Любой компонент DSL в системе **Endge** поддерживает универсальные атрибуты для задания **отступов**: `padding` и `margin`. Это позволяет гибко и быстро управлять расположением элементов без необходимости писать отдельные CSS-правила.

## Поддерживаемые атрибуты

| Атрибут | Описание                       | Пример               |
|---------|--------------------------------|----------------------|
| `p`     | Внутренние отступы (padding)   | `p="1"`, `p="0.5rem"` |
| `px`    | Горизонтальные паддинги        | `px="1"`             |
| `py`    | Вертикальные паддинги          | `py="0.5"`           |
| `pt`    | Верхний паддинг                | `pt="1"`             |
| `pb`    | Нижний паддинг                 | `pb="0.5"`           |
| `pl`    | Левый паддинг                  | `pl="1"`             |
| `pr`    | Правый паддинг                 | `pr="1"`             |
| `m`     | Внешние отступы (margin)       | `m="2"`              |
| `mx`    | Горизонтальные маргины         | `mx="1"`             |
| `my`    | Вертикальные маргины           | `my="0.5"`           |
| `mt`    | Верхний маргин                 | `mt="1"`             |
| `mb`    | Нижний маргин                  | `mb="0.5"`           |
| `ml`    | Левый маргин                   | `ml="1"`             |
| `mr`    | Правый маргин                  | `mr="1"`             |

⚙️ Значения **числовые** -  они умножаются на коэффициент `1rem` (по умолчанию).  
⚙️ Можно указывать и **любые CSS-единицы** напрямую (например, `"12px"`, `"1rem"` и т.д.).

## Пример использования

```jsx
<Text p="1" mx="2">
  Текст с отступами
</Text>
