# DSL Playground

**DSL Песочница** - плагин админки для интерактивной проверки JSX-разметки компонентов Endge.

## Назначение

Позволяет набирать и редактировать JSX-скрипт в редакторе и сразу смотреть результат рендера без сохранения в домен. Удобно для экспериментов с компонентами `<Text>`, `<Box>`, `<Flex>`, `<Layout>`, `<DateTime>`, условиями `if`/`else`, тултипами и стилями.

## Как пользоваться

- Открыть плагин через меню (например, «DSL Песочница»).
- В левой части - редактор JSX; в правой - превью. Соотношение панелей можно менять перетаскиванием разделителя.
- Доступны готовые примеры из документации (выпадающий список): Text, Box, Flex, Layout, If/Else, DateTime, Tooltip, стили.
- Содержимое редактора сохраняется в `localStorage` и восстанавливается при следующем открытии.
- Кнопка форматирования приводит JSX к единому стилю.

Результат рендерится через тот же движок, что и DSL-компоненты в домене, поэтому поведение совпадает с боевым.
