## Reactive Phased graph (NovaGraph)

**Реактивный фазовый граф (NovaGraph)** - это библиотека для создания реактивных графов, которая позволяет легко управлять иерархической структурой в проекции 
фаз (исполняемых функций системы).

**Фаза** - это функция системы, которая изменяет ее состояние в проекции некоторого *свойства*.

**Свойство** - это некоторое состояние узла системы в локальной и глобальной системе координат.
Локальное состояние изолировано и не зависит от внешнего мира. 
Глобальное - вычисляется исходя из родительского узла.

**Обновление** системы происходит в фазах, которые запускаются самой системой или внешним импульсом.
Фазы выполняются только по требованию, когда есть узлы, требующие обновления своих свойств.
